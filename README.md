# Payroll Management System

This project is a payroll management application developed in Python using a graphical user interface with Tkinter. The system was created as part of the Data Structures course at Universidad Nacional Abierta y a Distancia (UNAD).

## Author

Santiago Martínez Herrera  
Systems Engineering Student  
Universidad Nacional Abierta y a Distancia – UNAD

## Project Description

The application allows the registration of employee information and calculates the payroll payment according to the employee's job position and the number of days worked.

The system includes a login interface, an employee registration form, and a payroll report showing the calculated payment.

This project applies object-oriented programming concepts such as classes, attributes, methods, and data abstraction.

## Features

- Secure access using password authentication
- Employee data registration
- Automatic payroll calculation
- Payroll report generation
- Graphical user interface using Tkinter
- Input validation

## Technologies Used

- Python
- Tkinter (Graphical User Interface)
- Object-Oriented Programming
- Git and GitHub
